Simple Javascript Game

I made a game from the W3 Schools JavaScript tutorial and then made a few changes to it. First I made the background black to make it look dark. Then I added a border on the top and bottom with the same colors as the walls, so it looks like you are flying through a cave. After that, I made the 'player' white so they could still be seen and did the same with the score counter. Finally, I decided that a cave should have a ceiling so I added functionality to detect if the player was touching the roof of the cave, and stopped their momentum.
